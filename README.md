# PMLA Brand Guide

A single-page brand system for PMLA, built with plain HTML and CSS.

## Live Share URL

public page:

- https://theodor90.github.io/pmla/

## Project Files

- `index.html` - GitHub Pages entry page
- `PMLA Brand Guide.html` - Original source page
- `pmla.css` - Styling, tokens, layout, and motion


## Notes

- Typography uses Google Fonts (Bodoni Moda, Hanken Grotesk, Space Mono)
- The guide is designed to present logo, palette, typography, and visual direction in one page
