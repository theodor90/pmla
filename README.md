# PMLA Brand Guide

A single-page brand system for PMLA, built with plain HTML and CSS.

## Live Share URL

After GitHub Pages finishes deployment, your public page will be:

- https://theodor90.github.io/pmla/

## Project Files

- `index.html` - GitHub Pages entry page
- `PMLA Brand Guide.html` - Original source page
- `pmla.css` - Styling, tokens, layout, and motion

## Deploy on GitHub Pages

1. Open repository Settings > Pages.
2. Under Build and deployment, set Source to GitHub Actions.
3. Push to `main` (already configured) and wait for the `Deploy static site to GitHub Pages` workflow to finish.

## Run Locally

Open `index.html` in your browser.

## Notes

- Typography uses Google Fonts (Bodoni Moda, Hanken Grotesk, Space Mono)
- The guide is designed to present logo, palette, typography, and visual direction in one page
